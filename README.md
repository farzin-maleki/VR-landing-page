# VR-landing-page

A simple landing page for a virtual reality themed website built to practice front-end layout and styling techniques using HTML and CSS.

## Technologies Used
- HTML5
- CSS3

## Features
- Clean landing page layout
- Styled sections and typography
- Simple and modern interface

## Purpose of the Project
This project was created as part of my front-end development practice to improve my understanding of layout structure and CSS styling.
